# Haste & Slash
Пошаговая roug-like приключенческая игра
# How To Build
## Cmake
Скачайте репозиторий в любую директорию  
```git clone https://github.com/MichailS-000/Haste_n_Slash.git```  
Перейдите в директорию с исходным кодом  
```cd Haste_n_Slash```  
Вызовите cmake скрипт  
```cmake -S . -B ./out```
